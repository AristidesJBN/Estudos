1 - Escreva um programa que solicita ao usuário para inserir um número e verifica se é par ou ímpar. Exiba uma mensagem correspondente.

2 - Crie um programa que recebe o ano de nascimento de uma pessoa e calcula sua idade com base no ano atual. Em seguida, determine se a pessoa é maior de idade ou menor de idade.

3 - Escreva um programa que pede ao usuário para inserir um valor em uma unidade de medida (por exemplo, metros) e converte esse valor para outra unidade (por exemplo, pés) com base em uma escolha do usuário.

4 - Implemente uma calculadora de empréstimo que pede ao usuário para inserir o valor do empréstimo, a taxa de juros anual e o número de anos. Com base nesses dados, calcule o valor total a ser pago e exiba-o. OBS.: Procure pela fórmula de juros compostos.

5 - Crie um programa que pede ao usuário para inserir os comprimentos dos lados de um triângulo e determina se ele é equilátero (todos os lados são iguais), isósceles (dois lados são iguais) ou escaleno (todos os lados são diferentes entre si).
